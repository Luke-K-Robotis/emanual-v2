The threshold value for determining the Following Error Bit of [Moving Status(541)]. If the difference between [Position Trajectory(560)] and [Present Position(552)] is greater than this value, the Following Error Bit of the [Moving Status(541)] register will be set to '1'. If this value is '0', the Following Errors are disabled.

|   Unit    |      Range        |
|:---------:|:-----------------:|
| 1 [Pulse] | 0 ~ 2,147,483,647 |
