It is the present position value of DYNAMIXEL.
The range of Present Position(36) is 0 ~ 1,023 (0x3FF), and the unit is 0.29 [&deg;].

![](/img/dxl/dx/dx_series_goal_position.png)

> The picture above is the front view of DYNAMIXEL.


:::warning

**CAUTION** : If it is set to Wheel Mode, the Present Position(36) cannot be used to measure the moving distance and the rotation frequency.

:::
