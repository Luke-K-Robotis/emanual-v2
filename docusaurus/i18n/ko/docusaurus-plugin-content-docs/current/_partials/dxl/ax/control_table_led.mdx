LED를 ON/OFF 합니다.

| 값        | 설명                |
| :-------: | :------------------ |
| 0(기본값) | LED를 Off 시킵니다. |
| 1         | LED를 On 시킵니다.  |

:::note

**참고** : 장치의 상태(조건)에 따른 LED의 동작입니다.

|     상태      | LED 동작 |
|:-------------:|:--------:|
|     부팅      | 1회 점멸 |
|  공장 초기화  | 4회 점멸 |
|     알람      |   점멸   |
|   부트 모드   |   점등   |

:::
